YouTube Stream URL API

API Usage

This API extracts stream URLs from YouTube videos using yt-dlp.

Endpoint

GET /:video_id

Replace :video_id with a YouTube video ID or full YouTube URL.

Example Request

GET /dQw4w9WgXcQ

or

GET /https://www.youtube.com/watch?v=dQw4w9WgXcQ
注意: ストリームURLのキャッシュ期限は最長12時間です。それ以降は新しいURLを取得する必要があります。
