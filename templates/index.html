YouTube Stream URL API

API 使用方法

このAPIは、YouTube動画のストリームURLを抽出します。

エンドポイント

GET /:video_id

:video_idをYouTubeの動画IDまたは完全なYouTube URLに置き換えてください。

リクエスト例

GET /dQw4w9WgXcQ

または

GET /https://www.youtube.com/watch?v=dQw4w9WgXcQ

レスポンス

正常時: ストリームURLのみが返されます

https://rr1---sn-q0c7rn76.googlevideo.com/videoplayback?...

エラー時: エラーメッセージが返されます

Invalid YouTube video ID or URL
注意: ストリームURLのキャッシュ期限は最長12時間です。それ以降は新しいURLを取得する必要があります。
